Provisional API Stubs
=====================

.. warning::

   These inherited package areas have not yet undergone the docstring and
   example audit applied to PaDIS.  The module index is provided for
   orientation, not as a claim of complete API documentation.

Models
------

See ``LION/models`` for post-processing, unrolled, learned-regulariser,
plug-and-play, and diffusion model families.

Datasets
--------

See ``LION/data_loaders`` for 2DeteCT, LUNA16, walnut, and other adapters.

Optimisers and reconstructors
-----------------------------

See ``LION/optimizers`` and ``LION/reconstructors`` for the inherited training
and inverse-solver interfaces.  Their source docstrings remain authoritative
until dedicated pages replace this stub.
