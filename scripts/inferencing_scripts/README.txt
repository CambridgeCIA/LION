MODELNAME_test.py is a generalized inferencing script if you only
want to inference a single model rather than every model in the ALL_MODELS
folder. There are instructions as comments at the top for how to modify the file.

The rest of the scripts are meant to work with many different models.
before using them, you must update the dictionary linking class imports
to its string identifier. The identifiers are chosen by the naming con
vention of the files in the ALL_MODELS folder. Currently it is,
"MODELNAME_EXPERIMENTTYPE_INITIALIZER" for example "LPD_LowDose_FBP".


Lastly, make sure to update the file paths for saving the file generated by
save_all_raw.py (and general_stats.py if you did not keep the same CSV
folder conventions)
